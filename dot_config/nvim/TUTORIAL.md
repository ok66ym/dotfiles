# Neovim チュートリアル インデックス

このチュートリアルは4つのドキュメントに分かれている。
`<Leader>?` から番号で各ドキュメントを開ける。

| キー | 内容 |
|------|------|
| `<Leader>?1` | Vim 基本操作（モード・移動・編集・保存） |
| `<Leader>?2` | ファイル操作（neo-tree・Telescope・バッファ・分割） |
| `<Leader>?3` | コード操作（LSP・補完・検索・Git） |
| `<Leader>?4` | 環境操作（ドキュメント参照・tmux・Claude Code） |

---

## 推奨する学習順序

```
1 → 2 → 3 → 4 の順に進める。

1と2は Vim 初心者向けの基礎。
3はコーディングで毎日使う操作。
4はこの環境固有の操作（tmux・Claude Code との連携）。
```

## 各ドキュメントを開く

フロートウィンドウ内での操作:

| キー | 動作 |
|------|------|
| `j / k` | スクロール |
| `Ctrl+d / Ctrl+u` | 半ページスクロール |
| `/{検索語}` | ドキュメント内を検索 |
| `n / N` | 次 / 前の検索結果へ |
| `q` | 閉じる |

## 組み込みチュートリアル

Neovim 公式のインタラクティブチュートリアルも利用できる（英語）:

```
:Tutor
```

実際にキーを押しながら練習できる。
