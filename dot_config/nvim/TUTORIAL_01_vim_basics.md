モード・移動・編集・保存の基本を身につける。

`<Leader>` = `Space`  |  インデックスに戻る: `<Leader>?r`

---

## Step 1: モードを理解する（最重要）

Neovim には複数の「モード」がある。**現在のモードによってキーの意味が変わる。**
これが Vim/Neovim を学ぶ上で最も重要な概念。

| モード | 役割 | 見分け方（画面左下） |
|--------|------|---------------------|
| Normal | コマンドを実行する（デフォルト） | 表示なし |
| Insert | 文字を入力する | `-- INSERT --` |
| Visual | テキストを選択する | `-- VISUAL --` |
| Command | `:` コマンドを実行する | `:` が表示される |

### モードの切り替え

```
Normal → Insert  : i（カーソル前）/ a（カーソル後）/ o（下に新規行）/ O（上に新規行）
Insert → Normal  : jj または Esc
Normal → Visual  : v（文字）/ V（行）/ Ctrl+v（矩形）
Visual → Normal  : Esc
Normal → Command : :（コロン）
Command → Normal : Esc または Enter（実行）
```

### 練習 1-A: モードを切り替えてみる

1. 適当なテキストファイルを開く（`nvim 任意のファイル名`）
2. 画面左下に何も表示されていない = Normal モード
3. `i` を押す → `-- INSERT --` が表示される（Insert モード）
4. 「hello」と入力する
5. `jj` を押す → Normal モードに戻る（`-- INSERT --` が消える）
6. `:q!` + `Enter` → 変更を保存せずに閉じる

## Step 2: Normal モードでの移動

### 基本の移動（矢印キーは使わない）

| キー | 動作 |
|------|------|
| `h` | 左へ1文字 |
| `j` | 下へ1行 |
| `k` | 上へ1行 |
| `l` | 右へ1文字 |
| `w` | 次の単語の先頭へ |
| `b` | 前の単語の先頭へ |
| `e` | 単語の末尾へ |
| `0` | 行頭へ |
| `$` | 行末へ |
| `gg` | ファイルの先頭へ |
| `G` | ファイルの末尾へ |
| `{数字}G` | 指定行へ（例: `20G` で20行目） |
| `Ctrl+d` | 半ページ下へスクロール |
| `Ctrl+u` | 半ページ上へスクロール |
| `Ctrl+o` | 前のカーソル位置に戻る（ジャンプ後に便利） |

### 数字と組み合わせる

コマンドの前に数字を付けると繰り返せる:

```
5j   → 5行下へ
3w   → 3単語右へ
10G  → 10行目へ
```

### 高速ジャンプ（Flash）

```
s + 2文字  → 画面内の任意の場所にジャンプ
             2文字入力するとラベルが表示される
             ラベルのキーを押すとその場所に飛ぶ
```

### 練習 2-A

1. ファイルを開いた状態で:
2. `j` を5回押して5行下へ
3. `5k` で5行上へ（数字との組み合わせ）
4. `w` で単語を3つ進む
5. `0` で行頭へ
6. `$` で行末へ
7. `gg` でファイル先頭へ、`G` でファイル末尾へ
8. `s` + 2文字でジャンプを試す

## Step 3: テキストを編集する

### Insert モードへの入り方

| キー | カーソル位置 |
|------|-------------|
| `i` | カーソルの前から入力 |
| `a` | カーソルの後から入力 |
| `I` | 行頭から入力 |
| `A` | 行末から入力 |
| `o` | 下に新規行を作って入力 |
| `O` | 上に新規行を作って入力 |

### 削除（Normal モード）

| キー | 動作 |
|------|------|
| `x` | カーソル下の1文字を削除 |
| `dd` | 現在行を削除（クリップボードに入る） |
| `dw` | 単語を削除 |
| `d$` | カーソルから行末まで削除 |
| `d0` | カーソルから行頭まで削除 |

### 変更（削除してInsertモードへ）

| キー | 動作 |
|------|------|
| `cc` | 現在行を削除してInsertモードへ |
| `cw` | 単語を削除してInsertモードへ |
| `c$` | 行末まで削除してInsertモードへ |
| `r{文字}` | カーソル下の1文字だけ置換（Insertモードに入らない） |

### コピー・貼り付け

| キー | 動作 |
|------|------|
| `yy` | 現在行をコピー |
| `yw` | 単語をコピー |
| `y$` | 行末までコピー |
| `p` | カーソルの後に貼り付け |
| `P` | カーソルの前に貼り付け |

### アンドゥ・リドゥ

| キー | 動作 |
|------|------|
| `u` | 取り消し（アンドゥ） |
| `Ctrl+r` | やり直し（リドゥ） |

### 繰り返し

| キー | 動作 |
|------|------|
| `.` | 直前の操作を繰り返す |

### 練習 3-A: 基本編集

1. テキストファイルを開く
2. `gg` でファイル先頭へ
3. `o` で新しい行を追加して「test」と入力
4. `jj` で Normal モードへ
5. `dd` で追加した行を削除
6. `u` でアンドゥ（削除を取り消す）
7. `dd` で再度削除し、`p` で別の行に貼り付け

### 練習 3-B: 単語の変更

1. 単語の上にカーソルを置く
2. `cw` で単語を削除してInsertモードへ
3. 別の単語を入力
4. `jj` でNormalモードへ
5. `.` で直前の変更を別の場所で繰り返す

## Step 4: Visual モードで範囲選択

| キー | モード |
|------|--------|
| `v` | 文字単位選択 |
| `V` | 行単位選択 |
| `Ctrl+v` | 矩形（ブロック）選択 |

選択後の操作:

| キー | 動作 |
|------|------|
| `d` | 削除 |
| `y` | コピー |
| `c` | 削除してInsertモードへ |
| `>` / `<` | インデント追加 / 削除 |
| `gc` | コメントトグル |

### 練習 4-A

1. `V` で行選択モードへ
2. `j` で3行分選択を広げる
3. `>` でインデントを追加
4. `u` でアンドゥ
5. 再度 `V` → `j j` → `gc` でコメントアウト

---

## Step 5: 保存・終了

| コマンド | 動作 |
|---------|------|
| `:w` | 保存（閉じない） |
| `:wq` または `ZZ` | 保存して閉じる |
| `:q!` または `ZQ` | 保存せずに閉じる（変更を破棄） |
| `:q` | 変更がなければ閉じる |
| `:bd` | バッファを閉じる（ファイルは残る） |

> `:w` → 保存のみ（閉じない）を最もよく使う。
> 変更を破棄したい場合は `:q!`。

## Step 1〜5 の総合練習

1. `nvim sample.txt` でファイルを新規作成
2. `i` でInsertモードへ → 3行文章を書く → `jj` で戻る
3. `gg` で先頭へ → `yy` で1行コピー → `G` で末尾へ → `p` で貼り付け
4. `V` で2行選択 → `>` でインデント追加
5. `u` で取り消し
6. `/` で検索してみる（次のステップの先取り）
7. `:wq` で保存して終了
